2. Create new Table
	sudo fdisk /dev/sdb
		n (new partition)
		p (primary)
		1 (make it the first partition)
		- use default start and ends - 
		w (write the partition table to the disk)
		
3. Format the drive 
	sudo mkfs -t ext3 /dev/sdb1

4. Modify reserved space to 1% instead of 5% (optional) 
	sudo tune2fs -m 1 /dev/sdb1

4. Create mount-point 
	cd media
	sudo mkdir Data1
	sudo chown -R yex7906 Data1
	sudo chgrp -R yex7906 Data1
	sudo echo "/dev/sdb1    /media/Data1    ext3    defaults    0    2" >> /etc/fstab
